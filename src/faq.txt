



context api - можно создавать состояния отдельнно от компонентов, потом
использовать это состояние в тех компонентах, которым это состоянии нужно.
Также можно изменять это состояние из любого из компонетов.

Контекст позволяет передавать данные через дерево компонентов без необходимости передавать пропсы на промежуточных уровнях.


Hooks:

useContext()
